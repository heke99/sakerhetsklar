# Tillgänglighetsredogörelse — mall

Denna mall genereras/anpassas per kund och deployment. Den följer kraven för
tillgänglighetsredogörelser enligt lagen om tillgänglighet till digital
offentlig service (DOS-lagen) och EN 301 549.

## Tillgänglighetsredogörelse för Säkerhetsklar

**[Organisationsnamn]** står bakom denna digitala tjänst. Vi vill att så många
som möjligt ska kunna använda den. Detta dokument beskriver hur
Säkerhetsklar uppfyller kraven i EN 301 549 / WCAG 2.1 nivå AA, eventuella
kända brister och hur du kan rapportera problem.

### Hur tillgänglig är tjänsten?

Tjänsten är byggd för att uppfylla WCAG 2.1 AA:

- Fullständig tangentbordsnavigering med synlig fokusmarkering.
- Semantisk HTML med landmärken, rubriknivåer och tabellrubriker.
- Formulär med kopplade etiketter, felmeddelanden med `role="alert"` och
  beskrivande hjälptexter.
- Statusinformation förmedlas med text, inte enbart färg.
- Språkmärkning (`lang="sv"`).
